# Speech-To-Speech Model

## Installation

To install the required dependencies, run:

```bash
pip install -r requirements.txt
```

## Running the Models

To run and experiment with the models, use the provided Jupyter notebooks.

1. Start Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

2. Navigate to the `models/` directory and explore the following subdirectories for specific model implementations:
   - `models/omni_phi/`: Omni-Phi model pipeline and inference.
   - `models/SpeechT5/`: SpeechT5 fine-tuning and evaluation.
   - `models/SpeechT5WavLM/`: SpeechT5 with WavLM integration.
   - `models/UNet/`: UNet-based audio processing.
